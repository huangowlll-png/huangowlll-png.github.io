隐私政策 (Privacy Policy)

最后更新日期：2026年3月9日

本隐私政策描述了 mastertok（以下简称"我们"、"我们的"）如何收集、使用、披露和保护您在使用我们的 TikTok 数据分析应用（以下简称"本应用"）时的信息。使用本应用即表示您同意本隐私政策的条款。

1. 我们收集的信息

1.1 您提供的信息

  • 账户信息：当您注册本应用时，我们可能收集您的用户名、电子邮件地址和密码。
  • TikTok 账户信息：当您连接 TikTok 账户时，我们通过 TikTok API 访问您在授权范围内允许的信息，包括用户名、头像、公开视频数据等。
  • 联系信息：当您联系我们的客户支持时，您提供的通信内容。

1.2 自动收集的信息

  • 设备信息：设备类型、操作系统版本、唯一设备标识符、IP 地址。
  • 使用数据：您如何使用本应用的信息，包括访问的功能、使用时长、点击行为等。
  • 日志数据：当您访问本应用时，我们的服务器自动记录的信息，如访问时间、请求的页面、错误日志等。

1.3 通过 TikTok API 收集的信息

在您授权后，我们可能通过 TikTok API 访问以下类型的信息：

  • 您公开的视频内容及相关元数据
  • 视频表现数据（观看次数、点赞数、评论数、分享数）
  • 账户公开信息（用户名、头像、简介）

2. 我们如何使用您的信息

我们使用收集的信息用于以下目的：

  • 提供服务：运行和维护本应用，提供您请求的功能
  • 分析和改进：了解用户如何使用本应用，改进我们的服务和用户体验
  • 客户支持：回复您的询问和解决问题
  • 合规和安全：检测、预防和解决欺诈、安全或技术问题，遵守法律义务
  • 通信：向您发送与服务相关的通知、更新和促销信息（您可选择退出）

3. 信息的共享和披露

我们不会出售您的个人信息。在以下情况下，我们可能共享您的信息：

  • 服务提供商：与帮助我们运营本应用的第三方服务提供商（如云存储、分析服务）共享，他们受合同约束仅能为本隐私政策所述目的使用信息。
  • 法律要求：如果法律要求或出于善意相信为遵守法律程序、保护我们的权利、保护您或公众的安全所必需，我们可能披露信息。
  • 业务转让：在合并、收购或资产出售的情况下，您的信息可能作为转让资产的一部分。
  • 征得同意：在您明确同意的情况下。

4. 数据保留

我们仅在为实现本隐私政策所述目的所必需的期间内保留您的信息，除非法律要求或允许更长的保留期。通常：

  • 账户信息：在您保留账户期间
  • 分析数据：保留不超过 24 个月
  • 日志数据：保留不超过 6 个月

5. 数据安全

我们实施合理的技术和组织安全措施，以保护您的信息免遭未经授权的访问、使用或披露。但请注意，互联网传输或电子存储方法并非 100%安全，我们无法保证绝对安全。

6. 您的权利和选择

根据适用法律，您可能拥有以下权利：

  • 访问您的个人信息
  • 更正不准确的个人信息
  • 删除您的个人信息
  • 反对或限制处理您的个人信息
  • 数据可携带性

如需行使这些权利，请通过以下方式联系我们。

7. 第三方服务

本应用使用 TikTok API 来提供服务。您的 TikTok 数据的使用受 TikTok 的隐私政策和服务条款约束。我们建议您查看 TikTok 的隐私政策以了解其数据做法。

8. 儿童隐私

本应用不面向 13 岁以下的儿童。我们不会故意收集 13 岁以下儿童的个人信息。如果您发现我们可能收集了 13 岁以下儿童的个人信息，请立即联系我们，我们将采取措施删除该信息。

9. 隐私政策的变更

我们可能不时更新本隐私政策。变更将在本页面发布时生效。我们鼓励您定期查看本隐私政策以了解任何变更。您在变更后继续使用本应用，即表示您接受修改后的隐私政策。
